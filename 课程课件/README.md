# 课程课件

本目录收录了硬件兴趣组（SIGHW）各次课程的课件文件。课程主要面向硬件开发初学者，涵盖 Arduino 入门、Verilog 语法、FPGA 开发与仿真等内容。每节课件提供原始格式（.odp / .pptx）及 PDF 版本，部分课节拆分为多个 Part 分别讲解不同专题。

---

## 第一节：Arduino、Zyqn 入门

介绍 Arduino 与 Zynq 平台的基本概念与开发环境搭建，适合完全零基础的同学快速上手嵌入式硬件开发。

- [第一节：Arduino、Zyqn入门.pdf](第一节：Arduino、Zyqn入门.pdf)

---

## 第二节：Verilog 语法、CLab 远程开发

介绍 Verilog 硬件描述语言的基本语法以及通过 CLab 平台进行 Vivado 远程开发的流程。

- [第二节：Verilog语法、CLab远程开发.pdf](第二节：Verilog语法、CLab远程开发.pdf)

### Part 1：Verilog 讲解

系统讲解 Verilog 的模块结构、数据类型、组合逻辑与时序逻辑等核心语法，并通过示例帮助理解硬件描述与软件编程的区别。

- [第二节-Part1：Verilog讲解.pdf](第二节-Part1：Verilog讲解.pdf)

### Part 2：使用 Mac 进行 Vivado 远程开发

介绍如何在 macOS 环境下通过 SSH 与 X11 转发连接远程服务器，使用 Vivado 进行 FPGA 工程开发与综合。

- [第二节-Part2：使用mac进行vivado远程开发.pdf](第二节-Part2：使用mac进行vivado远程开发.pdf)

---

## 第三节：FPGA 仿真、简单 IP 核引入

介绍 FPGA 设计中的行为仿真方法以及如何在 Vivado 中引入简单的 IP 核，加快硬件设计迭代效率。

- [第三节：FPGA仿真、简单IP核引入.pdf](第三节：FPGA仿真、简单IP核引入.pdf)
- [第三节：FPGA仿真、简单IP核引入.webm（课程录像）](https://disk.pku.edu.cn/link/AAB950B9673FA44C4BA122DEDCF3B67E29)

### Part 1：开源 RTL 行为仿真

介绍使用开源仿真工具（如 Verilator、iverilog）对 RTL 代码进行行为级仿真的方法，包括测试平台（testbench）的编写与波形查看。

- [第三节-Part1：开源RTL行为仿真.pdf](第三节-Part1：开源RTL行为仿真.pdf)
