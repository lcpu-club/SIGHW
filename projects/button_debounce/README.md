### Button_Debounce 

#### 鉴于在led_run中我们已经介绍了基本的vivado设计流程，这里我们不再赘述。

#### 代码讲解(基于PYNQ-Z2)

- 按键抖动是指在按键被按下或者释放的过程中，由于机械结构的形变，导致电路信号出现多次变化的现象。这种现象会导致系统误判按键状态，产生错误的输入。
- 为了消除按键抖动的影响，我们定义一个button_reg, 用来存放当前按键的稳定状态！同时定义一个计数器，只要当按键信号button_in于button_reg不同时，计数器开始+1，只要button_in和button_reg相同，立刻将计数器归零，这样设计保证了只有当按键的信号和之前的稳定状态不同，且持续了一段时间(只要中间胆敢跳回来，立刻重新计数)，我们才认为按键状态发生了改变，并且度过了抖动的阶段，最终将button_reg和button_out更新为当前按键状态。
- 一般来说按键抖动的影响时间在50ms左右，我们的PYNQ-Z2时钟频率是125MHz，设计消抖时间是100ms，那么计数器需要达到12500000才行，和代码中的值一致。
